// 01 — Mission

La Mission

Dans une ère où l'automatisation redéfinit notre quotidien, la capacité d'une machine à percevoir son environnement est devenue un pilier technologique crucial. Le projet ASA Radar naît de cette conviction : un système de détection spatial capable de cartographier son environnement en temps réel, de traiter les données instantanément, et de les diffuser vers n'importe quel écran connecté sur la planète.

Nos motivations étaient de relever un défi d'ingénierie ambitieux en unifiant trois mondes : la mécanique de précision avec un servomoteur de balayage, l'électronique embarquée avec Arduino et le capteur ultrason HC-SR04, et la programmation web IoT avec Firebase Realtime Database et Web Serial API. Chaque discipline apporte une brique essentielle à l'édifice, et leur convergence crée un système bien plus puissant que la somme de ses parties.

Le Projet ASA est une plateforme complète capable de capturer des données physiques, de les traiter instantanément et de les diffuser sur le Cloud Firebase. N'importe quel appareil — téléphone, tablette, ordinateur — peut devenir un écran de contrôle synchronisé en direct. Le mode Maître permet de piloter le radar depuis l'Arduino connecté, tandis que le mode Spectateur permet à n'importe qui de suivre les données en live depuis le Cloud, sans aucune connexion physique.

// 02 — Architecture

L'Architecture

⚙

Arduino

Cerveau du système. L'Arduino pilote le servomoteur et lit les données du capteur ultrason, puis les transmet via le port série USB.

⚡

HC-SR04

Capteur ultrason de distance. Émet des impulsions sonores et mesure le temps de retour pour calculer la distance des obstacles avec précision.

↻

Servomoteur

Moteur de rotation précis qui balaye de 0 à 180 degrés, permettant au capteur de scanner l'environnement sur un demi-cercle complet.

☍

Web Serial API

Technologie navigateur permettant une communication directe entre le navigateur web et l'Arduino via câble USB, sans logiciel intermédiaire.

☁

Firebase RTDB

Base de données en temps réel de Google. Diffuse les données du radar vers n'importe quel appareil connecté, avec une latence minimale.

🌐

Interface Web

Visualisation immersive du radar en canvas HTML5. Affichage du balayage, des cibles détectées, des angles et distances en temps réel.
