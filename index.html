Le Projet ASA

Le Projet ASA est une plateforme IoT complète. Elle capture des données physiques via un capteur ultrason HC-SR04, les traite instantanément avec une carte Arduino et les diffuse sur le Cloud via Firebase.

Notre objectif : démontrer comment le web en temps réel peut étendre les capacités d'un capteur physique pour créer une interface de surveillance professionnelle accessible partout.
